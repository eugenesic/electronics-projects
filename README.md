# Power Electronics & EMC Portfolio

Приватное портфолио проектов по силовой электронике, электромагнитной совместимости (EMC) и системам управления батареями (BMS).

## О себе
Senior Power Electronics Engineer с более чем 10-летним опытом:
- Разработка и испытания на ЭМС (сертификационные тесты по стандартам)
- Проектирование зарядных устройств (в т.ч. для космических приложений)
- PCB design в Altium Designer
- Моделирование в LTSpice и MATLAB/Simulink
- Эксплуатация промышленных энергосистем (Газпром)

Текущий фокус: переход на remote / freelance в области renewable energy, battery storage, EV и embedded systems.

## Навыки
- **EMC/EMI**: испытания, анализ, проектирование фильтров
- **Power Electronics**: DC-DC, AC-DC, inverters, chargers
- **BMS**: моделирование, алгоритмы балансировки, SoC/SoH estimation
- **Embedded C**: STM32, ARM Cortex-M (в разработке)
- **Python**: PySpice, SciPy, NumPy, Pandas, Matplotlib для симуляций и анализа данных
- **Tools**: Altium Designer, LTSpice, MATLAB, Docker
